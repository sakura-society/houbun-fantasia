# houbun-fantasia

芳文幻想曲是一个2D RPG轻百日常类游戏，使用unity制作，以芳文社漫画为主的一个同人游戏

芳文ファンタジアは、芳文社の漫画を原作とし、unityで制作された2D RPGです。

此仓库使用**unity2021.3.2f1c1**制作

## 简介

芳文狂想曲（不是kirara！）正式开始制作啦！

因为是2DRPG，所以我们希望有会美术的小伙伴加入我们制作团队qwq

这个游戏是主打将芳文社出版的萌系漫画改编成游戏，不同于kirara狂想曲，芳文狂想曲是一个类开放，2D，RPG游戏，预计你可以在此游戏中跟其他人贴贴，也可以在slow loop中的海边喝着智乃泡的咖啡等等www

或者你也可以体验在本溪湖旁跟芝麻凛露营，或者跟海果一起造火箭，只要漫画中出现的东西，我们都会尽力去完成！

## 项目相关

此项目使用`Visual Scripting`来可视化编程

贴图分辨率暂时设置为`32x32`（可能会降到`16x16`）

此项目开启了`URP(通用渲染管线)`，所以可能会导致一些并不属于`URP`的预制件导入后出现显示问题，还请自己修改

### 文件目录

```bash
Assets
├─Characters  # 人物贴图
├─Scenes  # 场景
├─Scripts #脚本
│  ├─C# #C#脚本
│  └─Visual Scripting #可视化脚本
├─Settings  # 设置相关
│  └─Scenes # 设置->场景
└─Tilemap # 场景贴图
```

此结构为基本结构，可能会在此结构中有所调整
